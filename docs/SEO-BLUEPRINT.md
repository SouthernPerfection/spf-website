## Global SEO practices (applied every page)
- One `<h1>`, logical H2/H3 order, semantic landmarks, skip-link
- Unique title + meta description + canonical (→ production domain)
- Open Graph + Twitter cards
- schema.org JSON-LD: BreadcrumbList + page-appropriate Service/Organization; **FAQPage** where an FAQ exists
- Internal links: every page links UP to its money page and ACROSS to siblings; homepage + industries hub distribute authority
- Fast: shared external stylesheet, zero-JS nav, font-display swap
- robots.txt + sitemap.xml (regenerate on cutover)
