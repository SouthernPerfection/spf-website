FAQ

Returnable steel containers — answered.

Do you build collapsible steel containers?

Yes. We build collapsible and knock-down steel containers that fold flat for efficient return freight, then set up fast at the line.

Can you make wire mesh containers to our spec?

Yes. We fabricate custom wire mesh and steel containers to your part, load, and stacking requirements, built to print.

Are the containers stackable and returnable?

Yes. Our containers are engineered to stack and to run as returnable assets for many trips.

Need a returnable container?

Send a part or a print — we'll spec a container and a number.
